# Sistema de Control Escolar

Este proyecto es una propuesta inicial para un **Sistema de Control Escolar** dirigido a instituciones de nivel preparatoria. El objetivo es mostrar, a través de una página web, las funcionalidades y módulos que formarán parte de una futura aplicación de escritorio.

## **Estructura del Proyecto**

El proyecto incluye las siguientes secciones principales:
1. **Gestión de Estudiantes**:
   - Registro, edición y eliminación de estudiantes.
   - Información personal: nombre, edad, matrícula, grupo, etc.
   - Historial académico, incluyendo calificaciones y asistencia.

2. **Módulo de Aspirantes**:
   - Registro de nuevos aspirantes con datos básicos.
   - Seguimiento del proceso de admisión.

3. **Gestión Académica**:
   - Control de materias o asignaturas.
   - Generación de horarios por grupo o docente.
   - Registro y consulta de calificaciones por materia.

4. **Gestión Administrativa**:
   - Información del personal docente y administrativo.
   - Asignación de roles y permisos según el tipo de usuario.

5. **Control de Pagos**:
   - Registro y seguimiento de pagos de colegiaturas u otros conceptos.
   - Generación de recibos o comprobantes.

6. **Reportes y Análisis**:
   - Generación de reportes en PDF o Excel (calificaciones, asistencia, estados financieros, etc.).
   - Gráficas para visualizar estadísticas clave.

7. **Configuración**:
   - Personalización de datos como nombre de la escuela, logotipo, periodos escolares, etc.

## **Características Técnicas**

El proyecto incluye:
- **Frontend**:
  - HTML: Para la estructura de las páginas.
  - CSS: Para el diseño visual moderno y profesional.
  
- **Logo**:
  - Un logo ubicado en la esquina superior izquierda de todas las páginas, con un diseño consistente.

## **Cómo Navegar**

Este proyecto está dividido en las siguientes páginas:
- `index.html`: Página principal con un menú de navegación a todas las secciones.
- Páginas específicas para cada módulo:
  - `gestion_estudiantes.html`
  - `modulo_aspirantes.html`
  - `gestion_academica.html`
  - `gestion_administrativa.html`
  - `control_pagos.html`
  - `reportes_analisis.html`
  - `configuracion.html`

El diseño y funcionalidad básica están vinculados al archivo `styles.css`.

## **Futuro Desarrollo**

Este proyecto es la primera fase de desarrollo. Se prevé:
- Implementar funcionalidad dinámica mediante PHP.
- Desarrollar la versión de escritorio usando Java.
